### Запуск в режиме разработки: 
#### В корне проекта: 
    npm install
    npm run start
Это запустит сервер на http://localhost:5000
#### Переходим в папку client и запускаем в новом окне терминала клієнт:
     npm install
     npm run start
#### в браузере открываем:
    http://localhost:4200
